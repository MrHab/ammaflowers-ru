# AMMA Flowers RUB

Локальная версия прайса AMMA Flowers с расчетом в рублях.

- Источник снимка: https://darindakepstar-pixel.github.io/ammaflowers/
- Строк в снимке: 1066.
- Курс по умолчанию: 11.2925 ₽ за ¥, поле редактируется и может обновляться с CBR.
- Клиентский коэффициент по умолчанию: 1.15.
- Доставка: 150 000 ₽ за 33 коробки 100x40x20, то есть 4 545.45 ₽ за короб.
- Прайс цветов подтягивается из AMMA при открытии страницы и затем обновляется каждые 5 минут.
- В интерфейсе есть две вкладки: срезанные цветы и упаковка. Горшечные позиции скрыты.

Формула для каждой позиции:

```text
цена_короба_₽ = цена_банча_¥ * банчей_в_коробе * коэффициент * курс + доставка_за_короб
цена_стебля_₽ = цена_короба_₽ / (банчей_в_коробе * стеблей_в_банче)
```

Для позиций, где в упаковке указаны граммы (`500g/вяз`, `800G/扎`, `1000g/bun`), поле AMMA со стеблями трактуется как вес вязки. Для пересчета используется стандартная оптовая вязка 10 стеблей:

```text
стеблей_на_кг = 10 / (грамм_в_вязке / 1000)
цена_стебля_₽ = цена_короба_₽ / (банчей_в_коробе * 10)
```

Это приближенный расчет: фактическое число стеблей в 1 кг зависит от культуры, длины и толщины стебля.

Откройте `index.html` в браузере. Проект не хранит логин или пароль.
